## **🌟 서비스 소개**

- 'FUNCTION' 은 "**경매**"를 통해 상품을 거래하는 플랫폼입니다
- "긴장감 넘치는 경매의 스릴"울  추구하며,
    타임어택 경매 ⏳를 통해 경매가 종료되면, 당첨여부를 바로 확인 가능합니다
- 당첨된 후에는 **판매자와 채팅**을 통해 구매를 확정할 수 있습니다
